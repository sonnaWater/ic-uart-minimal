#### 🧮 數學基礎深化
```markdown
🔸 貝爾曼方程式
  ├── 📈 狀態價值函數: V(s) = E[Rt+1 + γV(St+1)|St = s]
  ├── 🎯 動作價值函數: Q(s,a) = E[Rt+1 + γQ(St+1,At+1)|St = s, At = a]
  └── 🔄 最優性條件: V*(s) = max_a Q*(s,a)

🔸 策略梯度定理
  ├── 📊 策略參數化: πθ(a|s)
  ├── 🎯 目標函數: J(θ) = E[∑γ^t rt]
  └── 🔄 梯度計算: ∇θJ(θ) = E[∇θ log πθ(at|st) Qπ(st,at)]
```
